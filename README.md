# lampOS

lampOS is a browser-based operating system built with a minimalistic vibe.

**Try it live: [lamp-os.vercel.app](https://lamp-os.vercel.app/)**

## What is this?

Lampos is a browser interface that strips away the noise. No cluttered tab bars, no bookmark toolbars fighting for space, no extensions menu staring at you — just a clean, minimal way to browse the web. It runs entirely in your browser at the link above, no installation required.

A web-based operating system built around minimalism. No cluttered windows or tabs. No unnecessary features. Just a clean and minimal way to do the things you need. Runs entirely in the browser by pressing the demo link.

## Why I built this

The project started as an experiment to see how close a web application could get to feeling like a real operating system. Over time, it evolved into a fully interactive desktop experience. Also, this is what an ideal OS would look like for me!

## How to use it

Go to **[lamp-os.vercel.app](https://lamp-os.vercel.app/)**

No installation or signup required. Works directly in the browser.
